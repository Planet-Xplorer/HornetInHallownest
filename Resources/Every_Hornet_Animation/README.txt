This is a folder containing EVERY HORNET ANIMATION. It contains the sprites for her HUD, cutscenes, the usage of her needolin, references to every possible animation she  can do, etc.

The names for the crests are simple enough. Here is a clarification on the file names for the crests to the in game names:

Crest Name |	Anim Name
Reaper     |	Hornet CrestWeapon Scythe Anim
Wanderer   |  	Hornet CrestWeapon Dagger Anim
Beast 	   |	Hornet CrestWeapon Warrior Anim
Witch 	   |	Hornet CrestWeapon Whip Anim
Architect  |	Hornet CrestWeapon Drill Lance Anim
Shaman 	   |	Hornet CrestWeapon Shaman Anim
Cloakless  |	Hornet Cloakless Anim
Cursed     | 	Hornet Cursed Anim

All of hornet's regular movement is found in the "Knight" folder. The names of each animation should be equivalent to what is referred to within the code of Silksong, named in folders. Within each folder is each sprite, numbered and ordered.
